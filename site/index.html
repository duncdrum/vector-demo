Quick start

Prerequisites: Docker and a browser.

Image: duncdrum/existdb:experimental

docker run -d --name exist-semantic \
    -p 8080:8080 -p 8443:8443 \
    duncdrum/existdb:experimental

If the name already exists: docker rm -f exist-semantic first.

Open eXide at http://localhost:8080/exist/apps/eXide/index.html. Save collection.xconf in the collection — eXide syncs it to the system config path on save.
